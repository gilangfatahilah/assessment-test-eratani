# User Register Card

A simple register card based on API: https://gorest.co.in/public/v2/users

## Tech Stack

**Typescript, React, Vite, Tailwind, React Hook Form, Zod**

## Run Locally

Go to the project directory

```bash
  cd my-project
```

Install depedencies

```bash
  npm install
```

Setup .env

```bash
  cp .env.example .env

  replace the API_ACCESS_TOKEN with yours
```

Run the program

```bash
  npm run dev
```

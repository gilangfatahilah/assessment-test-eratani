# CRUD Table

A simple table that displays a list of random user data, and have a several features like update, delete and search data.

## Tech Stack

**Typescript, React, Vite, Tailwind, Tanstack Table, Zustand**

## Run Locally

Go to the project directory

```bash
  cd my-project
```

Install depedencies

```bash
  npm install
```

Run the program

```bash
  npm run dev
```
